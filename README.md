# USSD TRANSFER - تطبيق Flutter

  واجهة رسومية لخدمات USSD لبنك فلسطين وجوال باي.

  ## المميزات

  - دعم بنك فلسطين (`*267#`) وجوال باي (`*110#`)
  - إخفاء نافذة USSD أثناء العمل
  - تخزين PIN مشفر + بصمة الإصبع
  - سجل العمليات (آخر 50 عملية)
  - اختيار الشريحة (SIM 1 / SIM 2)
  - دعم اختيار جهة الاتصال مباشرة

  ## متطلبات البناء

  - Flutter SDK 3.x+
  - Android SDK (minSdk 21+)

  ## خطوات البناء

  ```bash
  flutter pub get
  flutter build apk --release
  ```

  الملف الناتج: `build/app/outputs/flutter-apk/app-release.apk`

  ## ملاحظات مهمة

  1. بعد التثبيت، فعّل **خدمة إمكانية الوصول** للتطبيق يدوياً
  2. منح أذونات **الهاتف وجهات الاتصال** عند أول تشغيل
  3. اضبط رقم الشريحة من صفحة **الإعدادات** (0 للأولى، 1 للثانية)

  ## هيكل المشروع

  ```
  lib/
  ├── main.dart
  ├── models/         # نماذج البيانات
  ├── screens/        # الشاشات

  ├── parsers/        # محلل ردود USSD
  └── widgets/        # مكونات الواجهة
  ```
